# Bobba Express Documentation

## Project Overview
Bobba Express is a full-stack application built with React and Node.js/Express.

## Architecture
- **Client**: React + Vite frontend application
- **Server**: Node.js + Express backend API
- **Docs**: Project documentation and API specifications

## Getting Started

### Client Setup
```bash
cd client
npm install
npm run dev
```

### Server Setup
```bash
cd server
npm install
npm run dev
```
